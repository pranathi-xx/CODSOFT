# Task 1 – Mobile App Signup Flow

## Description
This task showcases a mobile app signup flow designed in Figma. The interface provides a simple and user-friendly experience for new users to create an account through a clean and modern design.

- Login screen

- Clean and intuitive UI

- User-friendly navigation
