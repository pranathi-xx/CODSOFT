# Task 2 – Email Marketing Template

## Description
This task showcases a responsive email marketing template designed in Figma. The template follows a clean and professional layout suitable for promotional emails, newsletters, and marketing campaigns.

## Features
- Modern and clean email layout
- Professional typography
- Promotional banner section
- Call-to-action (CTA) button
- Product/Image sections
- Responsive structure
- Footer with social media icons and contact details

## Tools Used
- Figma
